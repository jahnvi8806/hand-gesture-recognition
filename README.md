# Hand Gesture Recognition

Real-time hand gesture recognition using MediaPipe and JavaScript.

## Features

- Real-time webcam detection
- Multiple hand gestures
- Gesture logging
- Camera flip support

## Run

Open index.html in Chrome or run with Live Server.

## Tech Stack

- HTML
- CSS
- JavaScript
- MediaPipe Hands
